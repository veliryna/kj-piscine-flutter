# movie-list

This app includes the following features:

- Displaying the top rated movies with descending rating on the home page of the app.
- By tapping on a movie, a new route with more detailed information regarding the movie is displayed.
- Includes a search bar to search for movies by name. The search function must return movies whose name includes the entered string. For example, if "vatar" is searched, "Avatar" must be included in the response. The search function must work like SQL's ilike comparison.
